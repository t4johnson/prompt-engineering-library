**Key Load Markers:**
- ~78–83 miles
- 9–10 sessions
- 2 workouts
- 3 easy doubles
- Long run hits 18–20 mile range

**Workout Details:**
- Wed: 3 x 10:00 @ tempo (~threshold) with 2:00 jog recovery
- Sun: Short hill reps or 6 x 1' on/1' off fartlek for variety

**Upgrade Options:**
- Turn Wed into wave tempo or steady state (5–6 mi continuous)
- Add trail segment to Fri or Sun for variety

**Safety Triggers:**
- Flat legs >2 days = drop Sun workout
- High stress/sleep issues = cut a double
- Niggles = reduce Fri long run to 12–14 mi easy
